### 斐波那契数列
求斐波那契数列第n项的值，斐波那契数列定义如下
$$
F(n) = F(n-1) + F(n-2)
$$
其中：
$ F(0) = 0, F(1) = 1$
例如

$$F(3) = F(2) + F(1) = F(1)+F(0)+F(1) = 1+0+1 =2$$

常见解法
递归
```golang
func fib(n int){
  if n < 2 {
    return n
  }
  return fib(n-1)+fib(n-2)
}
```
时间复杂度$\theta{({\frac{1+\sqrt{5}}{2}}^n)}$

动态规划
```golang
func fib(n int){
  a := 0
  b := 1
  for ;n>0; {
    a, b = b, a
    b = a+b;
    n--
  }
  return a
}

时间复杂度$\theta{(n)}$

矩阵快速幂

我们首先要注意到上面的a和b实际上是在每次循环时做一个转换
$$
b \rightarrow a
b + a \rightarrow b
$$
